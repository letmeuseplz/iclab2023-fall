1. 硬體架構優化：資源共享與低功耗運算乘法器資源共用 (Hardware Sharing)：針對新增的 Equalization 功能，不使用昂貴的除法器，改以乘法運算達成 。運算順序優化：在卷積層與 Equalization 階段，採用 「先加再乘」 的運算架構，大幅減少乘法器元件數量，進而降低動態功耗 (Dynamic Power) 與電路面積。精準度控制：運算過程嚴格遵守 IEEE-754 浮點數格式，確保輸出誤差控制在 0.01 以下 。
2. 時脈閘控技術 (Integrated Clock Gating, ICG)閘控策略：利用 cg_en 訊號實作細粒度的時脈閘控，在非運算狀態下關閉各處理模組的時脈 。暫存器穩定性優化：特別針對 Buffer 與控制訊號 (Control Signals) 的重置 (Reset) 邏輯進行設計，避免 Reset 與 Clock Gating 邏輯衝突，確保資料更新在各種狀態切換下皆能維持正確性。高效能轉換：達 96% 的 Clock Gating 覆蓋率，顯著壓低動態功耗。
3. 驗證與調適 (Verification & Debugging)跨平台模擬驗證：經歷從 Vivado 邏輯驗證到 Verdi 波形追蹤的完整過程。除錯實錄：解決了因 Flag 邏輯判斷錯誤及 Register 位寬 (Bit-width) 設定不當導致的模擬失敗，確保功能在 Gate-level 模擬中 100% 正確 。形式驗證 (Formal Verification)：通過 JasperGold SEC 檢查，證明 Gated 與 Non-gated 兩版電路邏輯等價 。
4. 合成結果與環境優勢先進製程導入：採用 TSMC 90nm 製程進行合成，相較於傳統教學環境，大幅縮減了實體面積。數據表現：Area: 僅 445,035 (遠低於 7,000,000 門檻)。Power: 動態功耗成功壓至 426 uW 等級。。
